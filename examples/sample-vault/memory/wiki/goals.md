# Goals (sample persona)

1. **Recurring revenue** — turn one-off project work into something more
   predictable than hourly billing.
2. **Stay sharp** — keep learning so the consulting stays differentiated.
3. **Protect energy** — avoid the burnout cycle of stacking discovery calls on
   top of delivery.
